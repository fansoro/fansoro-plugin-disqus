# Morfy Disqus Plugin
Morfy disqus plugin.  

Set your `disqus_shortname` in `/plugins/disqus/disqus.yml`
```
disqus_shortname: name
```

And use this code in Morfy Templates to show Disqus comments:
```
{Disqus::comments()}
```
