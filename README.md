# Disqus
Fansoro disqus plugin.  

Set your `disqus_shortname` in `/plugins/disqus/disqus.yml`
```
disqus_shortname: name
```

And use this code in Fansoro Templates to show Disqus comments:
```
{Disqus::comments()}
```
